# cbrainfuck
A (brainfuck)[https://en.wikipedia.org/wiki/Brainfuck] interpreter, written in C for Linux.

## Installation
`github clone https://www.github.com/zabertooth/cbrainfuck`
`cd src`
`make install`
`cd ../srcbf`
`bf alphabet.bf`
