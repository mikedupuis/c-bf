# cbrainfuck
A (brainfuck)[https://en.wikipedia.org/wiki/Brainfuck] interpreter, written in C for Linux.

## Installation
* `git clone https://www.github.com/zabertooth/cbrainfuck`
* `cd cbrainfuck/src`
* `make install`
* `bf ../srcbf/alphabet.bf`
